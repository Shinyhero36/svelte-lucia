# Sveltekit + Lucia

Auth with Lucia on Sveltekit

## Getting Started

Install the dependencies...

```bash
pnpm i
```

Add your environment variables to `.env` file

```bash
cp .env.example .env
```

Start the development server...

```bash
pnpm dev
```

## Routes

- `/` - Home (Protected)
- `/login` - Login
- `/register` - Register
